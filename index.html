🔥 Connect Firebase — One-time Setup
5 minutes. Then works forever on any device.
1
Create Firebase Project
Go to console.firebase.google.com → Add Project → Name: alico-enterprises → Disable Analytics → Create Project
2
Add Web App + Copy Config
⚙️ Project Settings → Your Apps → </> Web → Register as alico-web → Copy the firebaseConfig values below
3
Enable Firestore
Build → Firestore Database → Create Database → Test Mode → Region: asia-south1 → Done
4
Enable Authentication
Build → Authentication → Get Started → Enable Email/Password and Google → Users tab → Add User → your email + password
5
Deploy on Netlify
Go to netlify.com → Deploys → Drag updated folder → Same link updates!
